# Modrinth Web Application

This project is a comprehensive web application that utilizes all of Modrinth's API features. It provides a user-friendly interface for managing user data, notifications, payouts, analytics, project management, version control, reports, threads, collections, and organizations.

## Features

- **User Management**: Read, write, and manage user email and data.
- **Notifications**: Create and manage user notifications.
- **Payouts**: Read and write payout information.
- **Analytics**: View analytics data related to user activity and project performance.
- **Project Management**: Create, read, write, and delete projects.
- **Version Control**: Manage project versions, including creating and deleting versions.
- **Reports**: Create, read, write, and delete reports.
- **Threads**: Read and write discussion threads.
- **Collections**: Manage collections, including creating and deleting them.
- **Organizations**: Create, read, write, and delete organizations.
- **Settings**: Set and save a Personal Access Token (PAT) for API requests.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd modrinth-web-app
   ```
3. Open `index.html` in your web browser to start using the application.

## Usage

- Use the navigation links on the main page to access different features of the application.
- Follow the instructions on the settings page to save your Personal Access Token (PAT) for API requests.
